# Ambassador 大使模式
改變及管理應用程式容器與外部世界的通訊方式

## 典型樣貌
![](https://docs.microsoft.com/en-us/azure/architecture/patterns/_images/ambassador.png)

## 優點
* 建置模組化
* 可重複使用的容器也具有內在價值
* 關注點的分離使容器更容易建立和維護
